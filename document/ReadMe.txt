■ 概要

ニコニコ動画の宣伝者をリストアップするツールです
	

====================================================================================================
■ 動作環境、確認環境

動作確認OS Windows7
開発環境 Visual Studio 2015 C#

====================================================================================================
■ 使い方

○ 手順

(1) 一番上のテキストボックスに、ニコニコ動画のURLを入力（sm,so,nmに対応）
(2) 「解析」ボタンを押します
(3) しばらくすると、宣伝者様の名前とコメントがリストに表示されます

○ 敬称のつけかた
	敬称をつけるのチェックボックスをONにし
	横のテキストボックスで「様」「さま」と入力します
	
○ 「リストの作り方」について
	以下4種から選択できます
		01_未加工
		02_同名を統合
		03_同名を統合(逆順)
		04_逆順
	
	未加工は、取得したデータをそのまま利用します
	
	同名を統合は、同じ名前の宣伝者名を1つに統合します
		そのさい、コメントが複数ある場合は改行でつなげて統合されます

		例）
			AAA    AAA
			BBB → BBB
			AAA

○ 「リストの表示」について
	以下2種から選択できます
		01_デフォルト
		02_名前 x[宣伝回数]
		
	02_名前 x[宣伝回数]、を選択した場合
		「リストの作り方」で「同名を統合」していた場合のみ有効です
		リストに表示される名前の後ろに半角スペースに小文字のx、
		そのあとに宣伝した回数が添えられます
		
		例）
			AAA    AAA x2
			BBB → BBB
			AAA

○ 「テキストの表示」について
	右のテキストボックスに出力される形式を選択できます
	
○ スクリプトについて
	それぞれ、以下の拡張子pyのスクリプトが実行されています
		リストの作り方 : script/01_list_create/
		リストの表示   : script/02_list_view/
		テキストの表示 : script/03_text_view/
	
	起動時に、ファイルを検索します
	IronPython(2.7)形式で、それぞれの処理を自前で作ることが可能です

○ 「再解析」ボタンについて
	敬称や、リストの作り方などの設定変更を適応したい場合や
	スクリプトを修正した場合、こちらを使ってください。
	取得済みのデータを使って解析します（＝通信しないので、早い）
