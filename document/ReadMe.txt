■ 概要

ニコニコ動画の宣伝者をリストアップするツールです
	

====================================================================================================
■ 動作環境、確認環境

動作確認OS Windows7
開発環境 Visual Studio 2015 C#

====================================================================================================
■ 使い方

○ 手順

(1) 一番上のテキストボックスに、smからはじまる動画のIDないし、URLを入力
(2) 「解析」ボタンを押します
(3) しばらくすると、大きなテクストエリアに、宣伝者様の名前が一覧で表示されます

○ リストアップ方法の変更
	コンボボックスでリストアップ方法を変更できます
	初期値は「重複する名前を削除」で、合わせて7種類あります
	
	重複する名前を削除
		AAA    AAA
		BBB → BBB
		AAA
		
	重複する名前は、続けてリストアップする
		AAA    AAA
		BBB → AAA
		AAA    BBB
		
	重複する名前は、x[数字] で表記する
		AAA    AAA x2
		BBB → BBB
		AAA
		
	重複削除せず(未加工)
		WebAPIから取得したままの宣伝者様名を出力します（敬称の追加は可能）
		重複削除などが行われません
	
	逆順、重複する名前を削除
		WebAPIの取得順は逆の、宣伝順(古い順)で
		重複する名前を削除、と同じ出力を行います
	
	逆順、重複する名前は、続けてリストアップする
		WebAPIの取得順は逆の、宣伝順(古い順)で
		重複する名前は、続けてリストアップする、と同じ出力を行います
		
	
	逆順、重複する名前は、x[数字] で表記する
		WebAPIの取得順は逆の、宣伝順(古い順)で
		重複する名前は、x[数字] で表記する、と同じ出力を行います
