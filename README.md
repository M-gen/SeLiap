ニコ動宣伝者リストアップ

===

ニコニコ動画の動画制作で  
「宣伝(広告)者様のリストアップ」を行う労力を軽減するツールです  

## 開発環境

Windows7 - VisualStudio2015 C#
Python 3.5.1 (バッチ処理用、ビルドには依存しない)

## 利用しているもの
+ (予定)IronPython
